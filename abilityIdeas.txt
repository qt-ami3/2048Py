dictionary:

  node:
    a block containing a value evenly divisible by 2.
  points:
    uses to buy abilites as you progress through levels.

added:
  bomb:
    blows up a node.

freeze:
  freezes a node for a stacking duration.

  vines(alt) freezes node for static duration.

pipe:
  a directional pipe that allows occupying nodes to traverse only one axis.

function block:
  allows use of python for scripting.
